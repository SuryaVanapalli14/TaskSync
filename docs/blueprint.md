# **App Name**: TaskSync

## Core Features:

- Landing Page: Visually appealing landing page with key benefits and clear CTAs.
- User Authentication & Registration: User registration with 'Requester' and 'Helper' roles, plus login functionality.
- Task Marketplace: Responsive card grid displaying available tasks with key details.
- Create Task Form: Form for users to post tasks with title, description, location, date, budget, and number of workers.
- AI Price Suggestion: Suggest the optimal price for a task based on description, location and time, by using mock data for weather, holidays, and worker availability. The tool displays an explanation for its decision.
- Task Detail Page: Display task details with images, descriptions, and worker lists.
- Analytics Dashboard: Dashboard to visualize task-related data using charts and stat cards.

## Style Guidelines:

- Primary color: A professional and calming light blue (#79B4D8) to inspire confidence and trust.
- Background color: Light desaturated blue (#E8F4FC) for a clean and airy feel.
- Accent color: A contrasting light purple (#B19CD9) to highlight key actions and elements.
- Headline font: 'Space Grotesk' (sans-serif) for headlines and titles, providing a modern, tech-forward aesthetic.
- Body font: 'Inter' (sans-serif) for body text, ensuring readability and a clean, professional look.
- Use 'lucide-react' icons for a consistent and modern look throughout the app.
- Responsive design to ensure a seamless experience across devices, leveraging ShadCN UI and Tailwind CSS.
- Subtle animations for transitions and interactions to enhance user experience.
